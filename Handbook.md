# AI Employee Handbook

This handbook contains operating instructions for the AI employee on how to process different types of requests.

## Processing Guidelines

### Meeting Summaries
When processing meeting summaries:
1. Extract key decisions made
2. Identify action items and assignees
3. Highlight deadlines and important dates
4. Summarize discussion points concisely
5. Format as structured notes with headers

### Task Lists
When processing task lists:
1. Categorize tasks by priority (High, Medium, Low)
2. Identify dependencies between tasks
3. Estimate time requirements if not specified
4. Group related tasks together
5. Flag any tasks that require human approval

### Email Threads
When processing email threads:
1. Identify the main topic/purpose
2. Extract important information and decisions
3. Note any requested actions
4. Identify key participants
5. Summarize the conversation flow

### Project Documents
When processing project documents:
1. Identify project goals and objectives
2. Extract timeline and milestones
3. Note resource requirements
4. Flag any risks or concerns
5. Summarize key findings or recommendations

## State Management

### When to Update Status to "planning-ready"
- When initial processing is complete
- When all required information has been extracted
- When action items are clearly identified

### When to Move to "Pending Approval"
- When processing is complete and ready for human review
- When decisions requiring human input are identified
- When the AI has completed its assigned tasks

## Quality Assurance

### Before Moving to "Pending Approval"
- Verify all information is accurate
- Ensure all action items are clearly defined
- Check that no important details were missed
- Confirm formatting is clean and readable

### Checklist for Completion
- [ ] Information extraction complete
- [ ] Action items identified and assigned
- [ ] Deadlines and milestones noted
- [ ] Dependencies identified
- [ ] Human-required decisions flagged
- [ ] Document formatted appropriately
- [ ] Quality review completed
